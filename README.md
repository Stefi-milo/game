# Game Arcade 🎮

A browser-based game arcade that can be run locally using Docker.

## Quick Start

Make sure you have Docker installed:

https://docs.docker.com/get-docker/

## Run the game:
## use the latest tag, not :latest

docker run --rm -p 8080:80 ghcr.io/stefi-milo/game:xxxx

## Open the localhost

localhost:8080


HOME ARCADE
